Omarchy
=======

This directory contains some config overrides for various components of Omarchy (ie. hyprland & lazyvim configs)

## Install

1. Either manually copy files to their appropriate `.config` directories or use stow / symlinking

-----

## Config Directory Mapping

1. `omarchy/hypr` -> `~/.config/hypr`
2. `omarchy/lazyvim/lua/config` -> `~/.config/nvim/lua/config`
2. `omarchy/lazyvim/lua/plugins` -> `~/.config/nvim/lua/plugins`
